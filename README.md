# 🧼💧 JooTime

> Плати → Стирай → Пользуйся чистой водой  
> Умная система для стиральных машин в общежитиях

---

## 🌍 О проекте

**JooTime** — это как автоматы с питьевой водой в городе,  
но только для **стиральных машин в общежитии** 🏢

Теперь, чтобы постирать одежду, пользователь:

👉 оплачивает через **Kaspi**  
👉 получает доступ к машине  
👉 использует **фильтрованную воду**

---

## ⚡ Идея

В городах уже есть системы с фильтрованной водой (типа "живая вода"),  
мы адаптировали эту идею под:

🧺 **стирку одежды в общагах**

---

## 🚫 Проблемы

- ❌ Нет системы оплаты за стирку  
- ❌ Кто-то стирает бесплатно, кто-то ждет  
- ❌ Вода плохого качества  
- ❌ Быстро ломаются стиральные машины  
- ❌ Нет контроля использования  

---

## ✅ Решение

JooTime внедряет:

- 💧 Фильтрованную воду для стирки  
- 📲 Оплату через Kaspi (QR / перевод)  
- 🔐 Доступ к машинке только после оплаты  
- ⏱️ Автоматическое управление временем  
- 📊 Контроль использования
